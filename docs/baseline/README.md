# Baseline Drafts

This directory contains source documents and the first structured DashDesign
project baseline.

## Files

- `baseline.schema.json`: JSON Schema for versioned cloud baselines.
- `baseline.v1.draft.json`: first draft extracted from the current source
  documents.

## Current Audience Mode

The source documents are mostly B-side partnership materials. The current
baseline intentionally transforms those materials into a to-C parent/student
poster baseline.

Rules:

- Keep course facts, educational values, age range, student outcomes, and visual
  production rules.
- Remove or block institution-operation language such as partnership,
  franchise, revenue, customer acquisition, tuition conversion, and teacher
  training.
- Use B-side facts only for traceability, not for image prompt injection.
- Publish only reviewed baseline versions; uploaded documents create drafts.
